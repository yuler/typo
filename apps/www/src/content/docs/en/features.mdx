typo is designed to streamline your writing workflow through intelligent, system-wide AI integration.

## Multi-Model Support

typo is completely model-agnostic. You can connect it to powerful cloud models like **DeepSeek**, or run everything locally using **Ollama** for maximum privacy and zero latency.

## Global Hotkeys

No matter what application you are working in—be it a web browser, code editor, or messaging app—you can invoke typo instantly using the global hotkey:

`Ctrl/Cmd + Shift + X`

## Intelligent Pasting

Input method compatibility can often be a headache across different operating systems. typo solves this by using a clipboard-based replacement strategy. It seamlessly copies the AI's refined response to your system clipboard and pastes it directly at your current cursor position, ensuring maximum compatibility across all apps.

## Customizable Prompts

You are not limited to a single rigid AI behavior. You can tailor typo's logic entirely by configuring custom system prompts and creating [prompt shortcuts](/docs/prompt-shortcuts) to handle specific tasks like translation, summarizing, or tone adjustment.
