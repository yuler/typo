typo utilizes the industry-standard **OAuth 2.0 Device Authorization Grant** ([RFC 8628](https://datatracker.ietf.org/doc/html/rfc8628)) flow to securely authenticate users without requiring you to enter credentials directly into the desktop application.

## Login Flow

Follow these simple steps to authenticate:

1. **Initiate Login**: Click the **Login** button within the typo desktop app. The app will generate a secure user code and automatically open your system's default web browser.
2. **Authorize via Browser**: In the newly opened browser window, log in to your account. You will be prompted to enter or confirm the user code displayed in the desktop app.
3. **Success**: Once authorized in the browser, the desktop app will automatically detect the approval and establish a secure, long-lived session. You are now ready to use typo!
