typo は Tauri + Vue のデスクトップアプリで、選択テキストを AI で改善し、カーソル位置に置き換えます。

## ダウンロード

最新版は [typo 公式ダウンロードページ](/jp/downloads) から入手できます。過去のビルドは [Releases ページ](/jp/releases) を参照してください。

<Tabs>

  <Tab label="macOS">

  1. `typo.dmg` を開きます。
  2. typo アイコンを **アプリケーション** フォルダにドラッグします。
  3. **アプリケーション** から typo を起動します。セキュリティの確認が出たら **開く** をクリックします。
  4. アカウントにサインインします。
  5. **システム設定** で typo の **アクセシビリティ** を有効にし、選択テキストの取得と処理後の置換を行えるようにします。
  6. カスタムショートカットを設定します。既定は `Ctrl + Shift + X` です。

  </Tab>

  <Tab label="Linux">

  1. ダウンロードした `typo.deb` を開きます。
  2. ソフトウェアインストーラーでインストールするか、ターミナルで `sudo dpkg -i typo_*.deb` を実行します。
  3. アプリメニューから typo を起動します。
  4. アカウントにサインインします。
  5. Linux Wayland では選択範囲を取得するためシステムショートカットの設定が必要です — [よくある質問 — Linux Wayland 互換性](./faq#linux-wayland-compatibility) を参照。
  6. カスタムショートカットを設定します。既定は `Ctrl + Shift + X` です。

  AppImage も利用できます。手順は [よくある質問 — Linux インストール（AppImage）](./faq#linux-installation-appimage) を参照してください。

  </Tab>

  <Tab label="Windows">

  1. `typo_x64-setup.exe` インストーラーをダブルクリックします。
  2. ウィザードに従い、スタートメニューまたはデスクトップのショートカットから typo を起動します。
  3. アカウントにサインインします。
  4. カスタムショートカットを設定します。既定は `Ctrl + Shift + X` です。

  </Tab>

</Tabs>
