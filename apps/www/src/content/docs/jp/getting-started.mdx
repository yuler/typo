typoは、AIを活用したテキスト最適化ツールとして機能する Tauri + Vue デスクトップアプリケーションです。選択したテキストをキャプチャし、AIで処理した後、カーソル位置のテキストをインテリジェントに置換します。

このページでは、リポジトリのルートからのローカル開発について説明します。アプリを使用するだけの場合は、[GitHub Releasesからコンパイル済みのインストーラーをダウンロード](https://github.com/yuler/typo/releases)してください。

## 前提条件

- [Node.js](https://nodejs.org/) (推奨バージョンについてはリポジトリの `.nvmrc` を参照)
- [pnpm](https://pnpm.io/)
- [Tauri 2](https://v2.tauri.app/start/prerequisites/) 向けのRustツールチェーンおよびプラットフォーム依存関係

## インストールと実行

リポジトリのルートから実行します：

```bash
pnpm install
pnpm desktop:dev
```

これにより、開発モード (Tauri + Vite) でデスクトップアプリが起動します。
