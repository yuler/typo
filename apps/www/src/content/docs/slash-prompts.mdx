In **Settings → Prompts**, you can define up to **five** slash prompts.

## Keys and values

Each shortcut includes:

- A **key** — e.g. `/tr:zh` or `/prompt`
- A **value** — the instruction sent to the model when triggered

## Trigger from selection

Either form works:

- **Leading** — the first line starts with `/command`
- **Trailing** — the last line starts with `/command`

## Template placeholders

In values you can use:

- `{{args}}` — arguments after the command on the same line, e.g. `/prompt translate lightly`
- `{{text}}` — selected text after the command line is removed

## Example

```text
/prompt Translate to Japanese in polite style

你好啊
```

The model receives the shortcut **value** with placeholders expanded from the selection above.
