Define up to **five** custom slash prompts in **Settings → Prompts**.

## Keys and values

Each shortcut has:

- A **key** — e.g. `/tr:zh` or `/prompt`
- A **value** — the instruction sent to the model when triggered

## Trigger from selection

Use either form:

- **Leading** — first line starts with `/command`
- **Trailing** — last line starts with `/command`

## Template placeholders

In shortcut values:

- `{{args}}` — text after the command on the same line, e.g. `/prompt translate lightly`
- `{{text}}` — selected text after the command line is removed

## Example

```text
/prompt Translate to Japanese in polite style

你好啊
```

The model receives your shortcut **value** with placeholders expanded from the selection above.
