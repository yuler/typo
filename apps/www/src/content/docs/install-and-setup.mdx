typo is a Tauri + Vue desktop app that refines selected text with AI and replaces it at your cursor.

## Download

Get the latest build from the [typo download page](/downloads). For older builds, see the [Releases page](/releases).

<Tabs>

  <Tab label="macOS">

  1. Open `typo.dmg`.
  2. Drag the typo icon into the **Applications** folder.
  3. Launch typo from **Applications**. If you see a security prompt, click **Open**.
  4. Sign in to your account.
  5. In **System Settings**, enable **Accessibility** for typo so it can read the selection and replace it after processing.
  6. Set a custom hotkey; the default is `Ctrl + Shift + X`.

  </Tab>

  <Tab label="Linux">

  1. Open the downloaded `typo.deb`.
  2. Install with your software installer, or in a terminal run: `sudo dpkg -i typo_*.deb`.
  3. Start typo from the app menu.
  4. Sign in to your account.
  5. On Linux Wayland, configure a system shortcut to capture the selection — see [FAQ — Linux Wayland compatibility](./faq#linux-wayland-compatibility).
  6. Set a custom hotkey; the default is `Ctrl + Shift + X`.

  You can also use the AppImage; see [FAQ — Linux installation (AppImage)](./faq#linux-installation-appimage).

  </Tab>

  <Tab label="Windows">

  1. Double-click the `typo_x64-setup.exe` installer.
  2. Follow the wizard, then launch typo from the Start menu or desktop shortcut.
  3. Sign in to your account.
  4. Set a custom hotkey; the default is `Ctrl + Shift + X`.

  </Tab>

</Tabs>
