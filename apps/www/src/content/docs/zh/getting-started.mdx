typo 是一个基于 Tauri + Vue 的桌面应用程序，充当基于 AI 的文本优化工具。它会捕获您选中的文本，使用 AI 进行处理，然后在光标处智能替换它。

本页面介绍了从仓库根目录进行本地开发的流程。如果您只是想使用该应用程序，请[从 GitHub Releases 下载预编译的安装程序](https://github.com/yuler/typo/releases)。

## 先决条件

- [Node.js](https://nodejs.org/) (推荐版本请参见仓库的 `.nvmrc`)
- [pnpm](https://pnpm.io/)
- 用于 [Tauri 2](https://v2.tauri.app/start/prerequisites/) 的 Rust 工具链和平台依赖项

## 安装和运行

在仓库根目录执行：

```bash
pnpm install
pnpm desktop:dev
```

这将在开发模式下启动桌面应用程序 (Tauri + Vite)。
