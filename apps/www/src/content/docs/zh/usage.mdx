typo 设计在全局快捷键后台运行，因此您可以在不离开编辑器或浏览器的情况下优化文本。

## 基本流程

1. **选择** 您想要优化的任何文本。
2. **按下** `Ctrl/Cmd + Shift + X` 以激活应用程序。
3. **等待** AI 响应，并让其自动替换选中的内容。

## 工作原理

1. **捕获**：获取当前选中的文本。
2. **AI 处理**：使用自定义提示词将其发送给 AI（可在设置中配置）。
3. **替换**：将 AI 的响应复制到剪贴板，然后将其粘贴在当前光标位置（使用剪贴板有助于避免输入法问题）。

## 模型

- 使用 **DeepSeek** 或其他支持的已配置云端模型。
- 在本地使用 **Ollama** — 确保 `ollama serve` 正在运行并已拉取模型。

请参阅 [常见问题 — Ollama](./faq#ollama) 以获取 API 链接和提示。
