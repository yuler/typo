typo 是一款 Tauri + Vue 桌面应用，用 AI 优化选中文本并在光标处替换。

## 下载

请前往 [typo 官方下载页面](/zh/downloads) 获取最新版本。如需历史版本，请参阅 [Releases 页面](/zh/releases)。

<Tabs>

  <Tab label="macOS">

  1. 打开 `typo.dmg`。
  2. 将 typo 图标拖入「应用程序」文件夹。

  <DocImage
    src={dragToApplications}
    alt="三步 macOS 安装示意图：在下载中打开 typo.dmg，将 typo 拖入应用程序，并从应用程序打开 typo.app。"
    caption="将应用拖入“应用程序”以安装 typo。"
  />

  3. 从「应用程序」中打开 typo。若出现安全提示，点击「打开」。

  <DocImage
    src={clickOpen}
    alt="macOS 安全确认提示，突出显示“打开”按钮。"
    caption="如果 macOS 要求确认，请选择“打开”。"
  />

  4. 登录你的账户。
  5. 在**系统设置**中为 typo 启用**辅助功能**，以便获取选中文本并在处理后替换。
  6. 设置自定义快捷键，默认为 `Ctrl + Shift + X`。

  </Tab>

  <Tab label="Linux">

  1. 打开下载的 `typo.deb`。
  2. 在软件安装器中完成安装，或在终端执行：`sudo dpkg -i typo_*.deb`。
  3. 从应用菜单启动 typo。
  4. 登录你的账户。
  5. 在 Linux Wayland 上需配置系统快捷键以捕获选区 — 见[常见问题 — Linux Wayland 兼容性](./faq#linux-wayland-compatibility)。
  6. 设置自定义快捷键，默认为 `Ctrl + Shift + X`。

  也可使用 AppImage，流程见[常见问题 — Linux 安装（AppImage）](./faq#linux-installation-appimage)。

  </Tab>

  <Tab label="Windows">

  1. 双击 `typo_x64-setup.exe` 安装程序。
  2. 按向导完成安装，从开始菜单或桌面快捷方式启动 typo。
  3. 登录你的账户。
  4. 设置自定义快捷键，默认为 `Ctrl + Shift + X`。

  </Tab>

</Tabs>

## 授权设备

登录后，typo 会在桌面应用中显示授权码。点击**打开浏览器**继续。

<DocImage
  src={clickOpenBrowser}
  alt="typo 桌面端授权对话框，提示点击“打开浏览器”按钮。"
  caption="从桌面授权对话框打开浏览器。"
/>

在浏览器中确认相同的授权码，以完成设备授权。

<DocImage
  src={clickApprove}
  alt="浏览器设备授权页面，提示点击“批准”按钮。"
  caption="批准匹配的授权码以授权该设备。"
/>
