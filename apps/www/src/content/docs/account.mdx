typo uses **OAuth 2.0 Device Authorization** ([RFC 8628](https://datatracker.ietf.org/doc/html/rfc8628)) so you never enter a username and password in the desktop app.

## Login flow

1. **Start login** — Click **Login** in typo. The app shows a user code and opens your default browser.
2. **Authorize in the browser** — Sign in to your account and enter or confirm the code shown in the app.
3. **Done** — After authorization, typo establishes a secure session automatically.
