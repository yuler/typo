typo uses the **OAuth 2.0 Device Authorization Grant** ([RFC 8628](https://datatracker.ietf.org/doc/html/rfc8628)) so you never type credentials into the desktop app.

## Login flow

1. **Start login** — Click **Login** in typo. The app shows a user code and opens your default browser.
2. **Authorize in the browser** — Sign in and enter or confirm the code shown in the app.
3. **Done** — After approval, typo establishes a secure session automatically.
