typo is a Tauri + Vue desktop application that acts as an AI-powered text refinement tool. It captures your selected text, processes it with AI, and intelligently replaces it at your cursor.

This page covers local development from the repository root. If you are just looking to use the app, [download the prebuilt installers from GitHub Releases](https://github.com/yuler/typo/releases).

## Prerequisites

- [Node.js](https://nodejs.org/) (see the repo `.nvmrc` for the recommended version)
- [pnpm](https://pnpm.io/)
- Rust toolchain and platform dependencies for [Tauri 2](https://v2.tauri.app/start/prerequisites/)

## Install and run

From the repository root:

```bash
pnpm install
pnpm desktop:dev
```

That starts the desktop app in development mode (Tauri + Vite).
